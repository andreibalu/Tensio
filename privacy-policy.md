# Tensio Privacy Policy

Tensio is local-first. The current MVP build stores blood pressure readings in an on-device SwiftData database that stays on the user's iPhone.

Tensio does not require an account, does not use an analytics SDK, does not sell data, and does not send readings to a Tensio server.

The current MVP build does not yet offer Apple Health sync, report export, or backup export. If a later build adds any of those features, Tensio will ask for the relevant permission in app and this policy will update with the new data flow.
